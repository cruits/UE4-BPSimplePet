# Simple pet - Learning AI BP

一个UE4制作的简单的宠物功能。

## 功能简介

宠物时刻跟随主人，在主人受到攻击后，立即找到敌人并在合适位置攻击。

-可多人模式

-简单的攻击互动


## 引擎

版本：UE4.22.3

需要开启 Environment Querying System

```
编辑器偏好设置->实验性功能->AI->Environment Querying System
```

## 打开方式

安装合适的引擎版本。

克隆或下载本功能，双击文件夹内 Simplepet.uproject 即可。
